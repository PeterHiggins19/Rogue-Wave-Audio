## Key Data Points Integrated

### Acoustic Engineering
- Total diffraction budget: 6.02 dB = 20·log₁₀(2)
- Cabinet contributions: G_H=3.21dB@144Hz, G_W=1.48dB@312Hz, G_D=1.33dB@348Hz
- Cortical boundary: PV/SST transition at 1.41 kHz
- BTL crossovers: 430 Hz (PV-dominant), 1500 Hz (boundary), 10 kHz (SST-dominant)
- Group delay target: <1 ms across crossover regions
- Measurement targets: ±2 dB on-axis, ±3 dB off-axis (±45°)

### Mathematical Framework
- T2R coherence: γ²_total(f) = γ²_tx · γ²_prop · γ²_room · γ²_cortex
- Target coherence: γ² ≥ 0.92 (≤ 0.08 dB total loss)
- Convergence criterion: RMSE ≤ 0.05 dB (TAF pipeline)
- Unity constraint: Σ contributions = 1.0 (Higgins Operator H1)

### Neuroscience/Psychoacoustics
- PV/SST ratio healthy: SST/PV ≈ 3.08
- After noise-induced hearing loss: ratio drops to 1.20
- Human Q optimal: 83 dB ± 6 dB
- JND (just-noticeable-difference): 0.25 dB @ 1 kHz
- Critical bandwidth: 12-point table (100Hz→90Hz BW, 15kHz→3000Hz BW)

### System Specifications
- BTL TF baseline: 1.46 Hz–48 kHz, magnitude -27.5 dB to +8.12 dB
- Baffle step: ~7.81 dB @ 32 Hz
- Damping factor: Qt = 0.600
- THD target: <0.5% @ 85 dB SPL across 20 Hz–20 kHz
