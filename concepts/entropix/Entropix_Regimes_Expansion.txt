## Phase-Based Methodology Templates
To avoid redundancy, templates are defined per Phase. Each regime uses its template with parametrized values. Scales add minor adjustments (e.g., gravity constant for Planetary).

1. **Stable Phase Template**
   - **Methodology**: Models predictable, oscillatory behavior using damped harmonic oscillator. Suitable for low-criticality systems. Steps: Initialize with input as position/velocity; integrate ODE; output average position damped by entropy.
   - **Equations** (Derived from Newton's laws, classical mechanics):
     - d²x/dt² + 2*zeta*omega*dx/dt + omega²*x = 0  (zeta = damping = Criticality/2; omega = frequency = 1 + EntropyProduction * (1 - Criticality))
     - Add Gaussian noise ~ N(0, EntropyBand_factor) where Divergent=0.1*e, High=0.05*e, Moderate=0.01*e, Low=0.001*e.
   - **Traceability**: Standard SHO equation (ref: Goldstein Classical Mechanics). Params ensure stability (low zeta).

2. **Metastable Phase Template**
   - **Methodology**: Logistic growth with potential bifurcation. For systems near tipping points. Steps: Normalize input; integrate; output equilibrium state.
   - **Equations** (Derived from population dynamics, Verhulst model):
     - dy/dt = r * y * (1 - y/K)  (r = growth = 2 + Criticality * EntropyProduction; K = carrying capacity = 10 / (1 - Criticality))
     - Stochastic term: + sigma * dW/dt (sigma = EntropyBand_factor * e).
   - **Traceability**: Logistic map extended to ODE (ref: May 1976 Nature). Criticality increases r towards chaos.

3. **Critical/Supercritical Phase Template**
   - **Methodology**: Chaotic attractor modeling for sensitive dependence. Steps: Use 3D state; integrate Lorenz; output std deviation as unpredictability measure.
   - **Equations** (Lorenz system, chaos theory):
     - dx/dt = sigma*(y - x); dy/dt = x*(rho - z) - y; dz/dt = x*y - beta*z
     - sigma = 10 * (1 + Criticality); rho = 28 * EntropyProduction; beta = 8/3 * (1 - Criticality)
     - Noise: Add to each eq ~ EntropyBand_factor.
   - **Traceability**: Original Lorenz 1963 (ref: Atmospheric convection). Modulated for criticality (higher rho → more chaos).

4. **Fragmenting Phase Template**
   - **Methodology**: Fragmentation/decay with branching. For breaking systems. Steps: Simulate exponential split; output fragment count or size distribution.
   - **Equations** (Derived from fracture mechanics/percolation):
     - dN/dt = lambda * N  (lambda = fragmentation rate = Criticality * EntropyProduction; N = fragments)
     - Power-law distribution: P(s) ~ s^(-tau) (tau = 2 + Criticality)
     - Stochastic breaks ~ Poisson(e).
   - **Traceability**: Percolation theory (ref: Stauffer 1994). Entropy drives branching.

5. **Collapsed Phase Template**
   - **Methodology**: Rapid decay to equilibrium. For post-critical collapse. Steps: Integrate decay; output residual energy.
   - **Equations** (Exponential decay, thermodynamics):
     - dy/dt = -gamma * y  (gamma = decay = EntropyProduction * Criticality)
     - Final y = y0 * exp(-gamma * t) + noise ~ EntropyBand_factor.
   - **Traceability**: First-order kinetics (ref: Chemical physics). High criticality accelerates collapse.
